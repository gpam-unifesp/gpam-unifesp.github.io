# GPAM Website Starter

Static bilingual website prepared for GitHub Pages.

## Structure
- English is the default version at the repository root.
- Portuguese pages are inside the `/pt` folder.
- Shared assets are inside `/assets`.

## Quick publish
1. Upload all files to the repository `gpam-unifesp.github.io`.
2. In GitHub, go to **Settings > Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and the `/ (root)` folder.

## Recommended first edits
- Replace placeholder email addresses on `contact.html` and `pt/contact.html`.
- Update member profiles on `people.html` and `pt/people.html`.
- Add official project descriptions on `projects.html` and `pt/projects.html`.
- Insert selected publications or external profile links.
- Replace placeholder images in `assets/img/people/` if desired.
